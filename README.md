# 📊 Shipping Cost Analysis Dashboard

Dashboard interaktif berbasis **Streamlit** untuk menganalisis biaya pengiriman (*freight cost*), hubungan antara harga produk dan ongkir, serta perbedaan biaya antar wilayah.
